# PassGuard - Password Manager

## Description

PassGuard is a secure, intuitive password manager built with the MERN stack (MongoDB, Express, React, Node.js). It provides a modern and responsive interface to safely store, organize, and retrieve your login credentials, helping you manage passwords efficiently and securely.

---

## Key Features

- **Secure Password Storage**: Safely save your website credentials with encryption.
- **Edit & Update Entries**: Modify stored passwords and related information with ease.
- **Delete Passwords**: Remove outdated or unnecessary entries permanently.
- **Copy to Clipboard**: One-click copy functionality for usernames, passwords, and URLs.
- **Password Visibility Toggle**: Show or mask passwords for enhanced security.
- **Responsive UI**: Built with Tailwind CSS to ensure a clean, modern user experience across devices.
- **Real-time Notifications**: Instant feedback and alerts powered by React Toastify.
